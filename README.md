## 🚀 Getting Started

### Prerequisites
- Node.js & npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Arsan-sk/My_Portfolio.git

# Navigate to the project directory
cd My_Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
